#  Library Management System in C

##  About the Project
This is a simple Library Management System made using C programming language.  
I created this project as part of my BCA 2nd semester.

The main idea of this project is to manage books in a small library using basic programming concepts. It uses linked list and dynamic memory allocation instead of arrays, which makes it more flexible.



##  Features
- Add new books  
- Display all books  
- Search book by ID  
- Issue book  
- Return book  
- Delete book  



##  Concepts Used
- C Programming  
- Structures  
- Pointers  
- Linked List  
- Dynamic Memory Allocation  



##  How to Run
1. Open the code in CodeBlocks or Dev C++  
2. Compile the program  
3. Run it  
4. Use menu options to perform operations  



##  Output
The program runs in console and shows a menu.  
User can choose options to manage books easily.



##  Purpose
This project helped me understand how data can be stored dynamically and how real-life systems like library work in programming.
